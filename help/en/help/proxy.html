Proxy/Firewall

To get to this panel, press button Settings in JonDo's main window, then choose the entry Proxy/Firewall.
Some users cannot use the anonymity service directly. This is the case if they are blocked by a firewall and thus cannot connect to the first mix. Furthermore, some internet providers require an obligatory proxy.

If you are certain that you are forced to use an obligatory proxy, you can change the configuration of the proxy conection here.

Proxy/Firewall

  • I'm forced to use a proxy
    Check this box to change the obligatory proxy settings.

  • Type
    Set the type of your proxy. Possibile types are "Socks-Proxy" and "HTTP/HTTPS-Proxy".

  • Host
    Enter the host name or IP address of your proxy here.

  • Port
    Enter the port number at which your proxy receives connections here.

  • Proxy requires authorization
    Check this box if you have to log in to your proxy.

  • User ID
    Enter your user name for connecting to the proxy here. The password will automatically be requested when you connect to the proxy for the first time.

If you have a firewall on your computer, please configure it so that JonDo can connect to the internet, specifically for the address of the InfoService and that of the first mix in the mix cascade.
