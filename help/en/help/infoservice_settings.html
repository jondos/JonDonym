InfoService

This panel is only available in Extended View.

To get to this panel, press button Settings in JonDo's main window, then choose the entry InfoService from the list to the left, then open the tab Settings.
The Info Service is used to provide JonDo with necessary information about the anonymity service. At the moment this includes:

  • The current JonDo version number to allow automatic updates.
  • The address of the first mix.
  • Information on the mix cascade, e.g. the public keys of the mixes.
  • Information on the service's capacity and the level of anonymity achieved by JonDo.
Please note that the correct information for the InfoService is already set by default. Changes to these settings could cause JonDo to no longer find the anonymization service.

Configuration

On this page, you can see the InfoServices that are known to your JonDo client. The currently preferred InfoService appears in bold. You can take the following actions:
  • Download list...
    Here you can download a new list of InfoServices from the internet.

  • Set as default
    By clicking this button, the selected InfoService will be chosen as the preferred InfoService

  • Certificate:
    A click on this button will show the certificate for this InfoService.

  • Add...
    When you choose this action, an additional dialog area for manually adding an InfoService appears below.

    • InfoService host
      Enter the host name or IP address of the InfoService that you want to add here.

    • InfoService port
      Enter the port of the new InfoService here.

    • InfoService name
      If you would like, you can enter a name for the InfoService here. The new InfoService then appears under this name in the list of known InfoServices. If this field is left empty, the InfoService is listed as a combination of host name and port.

    • Add
      Click this button to confirm your entry and insert the new InfoService.

    • Cancel
      With this button, the entries for the new InfoService will be discarded and you will return to the previous window.

    This part of the window also appears if you select an InfoService which has already been entered manually. You may then change this InfoService's data or delete it.

  • Remove
    When you click on this button, the selected InfoService will be removed from the list of known InfoServices.
