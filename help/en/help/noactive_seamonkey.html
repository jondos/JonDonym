Avoid data trails on the internet: Mozilla SeaMonkey

In order to avoid deanonymisation while browsing with SeaMonkey and JonDo, we need to do some settings. Open preferences by clicking the menu Edit -Preferences
  • Cookies
    Expand the item Privacy & Security in the list to the left and then choose Cookies. Select Allow all cookies and Accept for current session only. This way you won't have trouble with any sites since you accept them all, but you cannot be tracked.

  • Java
    Choose the item Advanced in the list to the left. Uncheck Enable Java.

  • JavaScript
    Choose the item Scripts & Plugins, which is a sub item of Advanced. It is recommended to disable JavaScript completely. In order to do so, uncheck Navigator.

Test: Test these settings for anonymity using this guide.

Note: These are basic settings. Using a filter proxy will offer more options for blocking content.
