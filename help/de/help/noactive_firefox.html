Vermeiden von Datenspuren im Internet: Mozilla Firefox

Sie können die Einstellungen für den Firefox manuell vornehmen oder ein Plugin einsetzen.

Manuell

Öffnen Sie im Menü Extras den Punkt Einstellungen.
  • Cookies
    Wählen Sie den Tab Datenschutz und haken Sie Cookies akzeptieren an. Wählen Sie bei Cookies behalten: die Option bis Firefox geschlossen wird. Dadurch können Sie alle Seiten besuchen, die zwingend Cookies voraussetzen, können jedoch durch Cookies nicht wiedererkannt werden.

  • Java & JavaScript
    Wählen Sie den Tab Inhalt. Haken Sie Java aktivieren und JavaScript aktivieren ab.
Hinweis: Das sind lediglich grundlegende Einstellungen. Die Verwendung eines Filterproxys ermöglicht mehr Einstellungen zum Blockieren von Inhalten.
Das Plugin Cookie Culler ermöglicht das Beseitigen von unerwünschten sowie das Verwenden von gewünschten Cookies.

Plugin: Verwenden von PrefBar

Wenn Sie PrefBar zum komfortablen Verwalten der Einstellungen verwenden wollen, lesen Sie bitte diese Beschreibung.

Test: Unterziehen Sie diese Einstellungen einem Anonymitätstest.
