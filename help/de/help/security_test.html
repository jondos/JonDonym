Einstellungen des Browsers / Filters testen

Im folgenden verwenden wir Testseiten, die uns erkennen lassen, ob der Browser (und, soweit vorhanden, der Filterproxy) richtig konfiguriert wurde. JonDos empfiehlt den bereits fertig konfigurierten, portablen JonDoFox zum sicheren, anonymen Websurfen. Er ist als vollständiger Browser (Windows) oder als Firefox-Profil für alle Betriebssysteme verfügbar und besteht alle Tests.

JonDos-Anonymitätstest

Die Seite gibt einen schnellen Überblick über alle IP-, Header- und sonstigen Informationen, die Ihr Browser an Webseiten sendet. Alle Felder sollten grün gefärbt oder farblos sein, Orange wäre tolerierbar, Rot könnte verheerend für Ihre Anonymität sein. Wenn Sie Orange oder Rote Farben sehen, bewegen Sie den Mauszeiger über die jeweiligen Felder. Sie werden dann Hinweise und Erklärungen als Popups erhalten.

Sollten die Testergebnisse nicht den beschriebenen Anforderungen entsprechen, so kann das u.a. folgende Ursachen haben:

  • JonDo ist nicht korrekt eingerichtet
  • Sie habe die Datenschutzeinstellungen im Browser nicht korrekt eingestellt
  • Es wird kein Filterproxy verwendet.
  • Der Filterproxy verwendet nicht die richtigen Filter.
  • Der Filterproxy benutzt nicht JonDo als externen Proxy.
  • Der Browser benutzt nicht den Filterproxy als Proxy.
Weitere Testseiten:
  • Leader.ru Überblick über verschiedene Header
  • Gurushaven.de Sehr ausführliche Beschreibungen von verschiedenen Tests
  • WhatIsMyIP Zeigt die externe IP Adresse an.
  • Astalavista Zeigt Verbindungsinformationen an (HTTPS)
  • Proxyjudge Versucht, eine Aussage über das Anonymitätslevel zu machen (irrelevantes Bewertungsschema)
  • RRDB Zeigt Verbindungsinformationen an
  • NetReal Deutsche Seite mit guten Erklärungen
  • Browserspy Enthält eine große Liste mit Identifizierungsmerkmalen
