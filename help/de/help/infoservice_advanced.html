InfoService: Erweiterte Einstellungen

Dieses Panel steht nur in der erweiterten Ansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen, wählen anschließend aus der linken Liste den Eintrag InfoService und öffnen anschließend den Tab Erweiterte Einstellungen.

  • Automatische InfoService-Anfragen durchführen
    Der JonDo versucht an einigen Stellen automatisch eine Verbindung zum InfoService aufzubauen und Abfragen durchzuführen. Wenn dieses Feld nicht markiert ist, unterbleiben diese automatischen Abfragen.

  • Nicht-Anonymisierte Verbindungen zu InfoServices erlauben
    Ist diese Option aktiviert, kann JonDo Verbindungen zu den InfoServices herstellen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können nur über anonymisierte Verbindungen Informationen von den InfoServices abgerufen werden.
    Es wird deshalb empfohlen, diese Option zu aktivieren.

  • Redundante Anfragen an verschiedene InfoServices stellen
    Legen Sie fest, von wievielen InfoServices Informationen geholt werden sollen. Legen Sie z.B. als Anzahl "3" fest, so wird JonDo immer versuchen, 3 InfoServices zu erreichen. Fällt einer aus, wird versucht einen anderen als Ersatz zu erreichen. Je mehr InfoServices Sie festlegen, desto robuster wird JonDo gegen Störungen in den InfoServices, desto länger dauert die Abfrage aber auch.
    Entwicklerversion: Es wird empfohlen, diese Option zu aktivieren.

  • Verbindungs-Timeout
    Sollten Ihre Internetanbindung zu langsam sein, so kann ein zu geringes Timeout bei der Abfrage der InfoService-Informationen dazu führen, dass keine Verbindung aufgebaut wird und sie keine Informationen über Dienste, Updates, InfoServices usw. erhalten. Ein zu hohes Timeout führt unter Umständen zu hohen Wartezeiten beim Abruf der Informationen, wenn ein oder mehrere abgefragte InfoServices zu langsam oder nicht verfügbar sind. Stellen Sie das Timeout deshalb bei Bedarf höher oder niedriger ein.
