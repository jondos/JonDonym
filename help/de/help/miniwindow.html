Die JonDo Oberfläche: Das Mini-Fenster

Das Mini-Fenster von JonDo ist eine minimalistische Ausgabe des Hauptfensters. Sie können es durch einen Klick auf oder durch einen Doppelklick in das Hauptfenster aufrufen. Dieses Fenster zeigt nur die wichtigsten Informationen an und besteht aus folgenden Elementen:

  • Bytes: Zeigt, wieviele Daten während dieser Sitzung schon anonymisiert wurden.
  • Anon: Stellt den Grad der Anonymität dar. Die angezeigten Werte entsprechen jeweils der Darstellung des Anonym-O-Meters.
  • zum Ein- und Ausschalten der Anonymität.
  • Knopf, der zur normalen Darstellung von JonDo umschaltet (geht auch mit einem Doppelklick in das Fenster).

Kontextmenü

Sie können über das Kontextmenü von JonDo immer schnell auf die wichtigsten Funktionen zugreifen. Das Kontextmenü ist durch einen Klick mit der rechten Maustauste auf das Hauptfenster, das Mini-Fenster oder das kleine Icon in der Windows-Schaltfläche erreichbar.
