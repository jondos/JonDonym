Kontextmenü

Sie können über das Kontextmenü von JonDo immer schnell auf die wichtigsten Funktionen zugreifen. Das Kontextmenü ist durch einen Klick mit der rechten Maustauste auf das Hauptfenster, die Mini-Ansicht oder das Icon in der Windows-Schaltfläche erreichbar.
