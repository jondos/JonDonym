Einspielen von Updates

Um zu diesem Einstellungsfenster zu gelangen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Update. Dieses Einstellungsfenster steht Ihnen nur in der erweiterten Ansicht zur Verfügung.

In diesem Dialog können Sie prüfen, ob neue Versionen von JonDo zum Einspielen verfügbar sind.

Installierte Version

  • Version
    Hier sehen Sie die Versionsnummer des JonDos.

  • Datum
    Dieses Feld zeigt Ihnen den Zeitpunkt, an dem die momentan installierte Version veröffentlicht wurde.

  • Art
    Es gibt zwei verschiedene Versionsarten: Die stabile Version wurde ausgiebig getestet. Die Beta-Version, auch Entwicklerversion genannt, enthält alle derzeit verfügbaren Funktionen, die sich aber noch in der Testphase befinden. Aus diesem Grund ist die Beta-Version eher geeignet für erfahrene und experimentierfreudige Benutzer.

Aktuelle Version

Die hier angezeigten Informationen beziehen sich aber nicht auf die auf Ihrem System installierte Version, sondern auf diejenige, die momentan zum Download zur Verfügung steht. Mit Hilfe der Auswahlbox Art können Sie festlegen, ob Sie zur Stabilen Version oder zur Entwicklerversion umschalten möchten.

Optionen

  • Nicht-anonymisiertes (direktes) Update erlauben
    Es wird empfohlen, diese Option zu aktivieren. Dadurch kann JonDo ein Update vornehmen, auch wenn Sie nicht mit einem JonDonym-Dienst verbunden sind. Wird diese Option nicht aktiviert, können Updates nur über eine anonymisierte Verbindung erfolgen. Dadurch kann es sein, dass Sie nicht immer Zugriff auf die aktuelle Version erhalten.

  • Hinweisfenster anzeigen, wenn neue JAP/JonDo-Version verfügbar ist
    Liegt eine neue Version von JonDo vor und haben Sie diese Option aktiviert, erscheint ein Hinweisfenster mit der Möglichkeit, das Update vorzunehmen. Wenn diese Option deaktiviert ist, wird in der Statuszeile im Hauptfenster nur noch ein kleiner Hinweis angezeigt. Die Prüfung auf Updates erfolgt beim Start des Programms und danach alle 12 Stunden.

  • Hinweisfenster anzeigen, wenn neue Version von Java (JRE) verfügbar ist
    JonDo prüft beim Start und danach alle 12 Stunden, ob eine neuere JAVA-Version verfügbar ist. Haben Sie diese Option aktiviert, erscheint ein Hinweisfenster. Eine Aktualisierung der JRE muss manuell erfolgen. Mehr...

Info

In diesem Feld werden Ihnen Informationen über den Fortschritt beim Installieren eines Updates angezeigt. Außerdem zeigt es das Ergebnis der Prüfung auf neuere Versionen.

Weitere Bedienelemente

  • Prüfe, ob neue Updates vorhanden sind
    Wenn Sie diesen Knopf anklicken wird geprüft, ob eine neuere Version als die bei Ihnen installierte zum Download zur Verfügung steht.

  • Upgrade
    Hiermit können Sie die neueste Version von JonDo gemäß Ihrer Auswahl unter Art aus dem Internet laden und auf Ihrem System installieren.
