Einspielen von Updates

Dieses Panel steht Ihnen in der erweiterten Ansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Update. In diesem Dialog können Sie prüfen, ob neue Versionen von JonDo zum Einspielen verfügbar sind.

Installierte Version

  • Version:
    Hier sehen Sie die Versionsnummer der derzeit auf Ihrem System installierten Version von JonDo.

  • Datum:
    Dieses Feld zeigt Ihnen den Zeitpunkt, wann die momentan installierte Version veröffentlicht wurde.

  • Art:
    Hier wird angezeigt, um welche Art es sich bei der installierten Version handelt. Es gibt zwei verschiedene Versionsarten: Die "stabile Version" wurde ausgiebig getestet und enthält nur Funktionen, von denen angenommen wird, dass sie keine Fehler enthalten. Die "Entwicklerversion" enthält alle derzeit verfügbaren Funktionen, auch wenn diese nur teilweise implementiert oder fehlerhaft sind.
    Für Normalbenutzer wird empfohlen, die "stabile Version" zu verwenden.

Aktuelle Version

In dieser Gruppe sind die gleichen Felder wie in der Gruppe "Installierte Version" vorhanden. Die Informationen in den Feldern entsprechen den Informationen in der Gruppe "Installierte Version". Die hier angezeigten Informationen beziehen sich aber nicht auf die auf Ihrem System installierte Version, sondern auf die Version von JonDo, die momentan herunter geladen werden kann.
Mit Hilfe der Auswahlbox "Art" können Sie festlegen, ob Sie Informationen über die "Stabile Version" oder über die "Entwicklerversion" wünschen.

Optionen

  • Nicht-anonymisiertes Update erlauben
    Es wird empfohlen, diese Option zu aktivieren. Dadurch kann JonDo ein Update vornehmen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können Updates nur über eine anonymisierte Verbindung erfolgen.

  • Hinweisfenster anzeigen, wenn neue JonDo-Version verfügbar ist
    Liegt eine neue Version von JonDo vor und haben Sie diese Option aktiviert, erscheint ein Hinweisfenster mit der Möglichkeit, das Update vorzunehmen.
    Wenn diese Option deaktiviert ist, wird im Hauptfenster von JonDo dennoch ein kleiner Hinweis angezeigt. Die Prüfung auf Updates erfolgt beim Starten von JonDo und danach alle 12 Stunden.

  • Hinweisfenster anzeigen, wenn neue Version von Java (JRE) verfügbar ist
    JonDo prüft beim Start und danach alle 12 Stunden, ob eine neuere JAVA-Version verfügbar ist. Haben Sie diese Option aktiviert, erscheint in diesem Fall ein Hinweisfenster. In jedem Fall erscheint zusätzlich eine Markierung im Hauptfenster. Eine Aktualisierung der JRE muss manuell erfolgen. Mehr...

Info

In diesem Feld werden Ihnen Informationen über den Fortschritt beim Installieren eines Updates angezeigt. Ausserdem zeigt es das Ergebnis der Prüfung auf neuere Versionen.

Weitere Bedienelemente

  • Prüfe, ob neue Updates vorhanden sind:
    Wenn Sie diesen Knopf anklicken wird geprüft, ob sich aus dem Internet eine neuere Version als die derzeit bei Ihnen installierte herunterladen lässt.

  • Upgrade:
    Hiermit können Sie die neueste Version von JonDo aus dem Internet laden und auf Ihrem System installieren.
