Bezahlung

Um zu diesem Einstellungsfenster zu gelangen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Bezahlung.

In diesem Dialog können Sie neue Konten anlegen um die Premiumdienste nutzen zu können. Desweiteren haben Sie hier die Möglichkeit, Ihre bereits bestehenden Konten verwalten und alle weiteren Einstellungen vorzunehmen die Sie benötigen, um die Premiumdienste zu nutzen.

Konten

In der Liste sehen Sie die Konten, die Sie bereits angelegt haben. Wenn Sie diesen Dialog zum ersten Mal aufrufen, ist die Liste leer, da Sie noch kein Konto angelegt haben.

Neben der Liste mit den Konten sehen sie das folgende Informationen zu Ihrem Konto:

  • Erstellungsdatum
    gibt an, wann an welchem Datum das Konto erzeugt wurde.

  • Gültig bis
    Bis zu diesem Datum können Sie das entsprechende Konto nutzen.

  • Bezahlinstanz
    Bei diesem Anbieter wurde das Konto erzeugt. Darunter ist ein Verweis zu den geltenden Allgemeinen Geschäftsbedingungen angegeben.

Wenn neben der Liste der Hinweis "Keine Sicherheitskopie vorhanden" angezeigt ist, haben Sie von dem gewählten Konto noch keine Sicherheitskopie erstellt. Wir empfehlen Ihnen, von jedem Ihrer Konten eine Sicherung anzulegen. Dadurch ist Ihr Guthaben vor Datenverlust geschützt. Durch einen Klick auf den Hinweis oder des Buttons Exportieren haben Sie die Möglichkeit das ausgewählte Konto zu sichern.

Sie können folgende Aktionen durchführen:

  • Konto erzeugen
    Hiermit legen Sie ein neues Konto an. Der Kontoerstellungsprozess kann etwas Zeit in Anspruch nehmen, da hierfür ein aufwändiger kryptographischer Vorgang gestartet werden muss. Dies ist notwendig, damit die Kontoerzeugung vertraulich erfolgen kann und Ihnen keine gefälschten Informationen übermittelt werden. Während des Erstellungsvorgangs werden Sie möglicherweise aufgefordert, eine Sicherheitsfrage zu beantworten. Der Ablauf der Kontoerzeugung unterscheidet sich je nach Bezahlmethode. Bitte folgen Sie dazu den Anweisungen am Bildschirm. Wenn Ihr Konto erfolgreich erstellt wurde, erscheint es in der Liste der verfügbaren Konten.

    WICHTIG
    Bitte sichern Sie Ihr Konto, nachdem Sie es erzeugt haben. Ihr Guthaben geht unwiederbringlich verloren, falls die Konfigurationsdatei des JAPs gelöscht wird. Am sinnvollsten ist es, die Kontodatei auf einem externen Wechelsdatenträger, z.B. einem USB-Stick zu sichern. Falls es zu einem Datenverlust auf Ihrem System kommt, kann auch eine lokal abgespeicherte Kontodatei verlorengehen.

  • Transaktionen
    Hier können Sie sich die letzten Zahlungsvorgänge auf Ihre Konten ansehen.

    Hinweis
    Beachten Sie, dass für ein Konto jeweils nur eine Transaktion möglich ist. Sie können also ein leeres Konto löschen und wenn Sie es wünschen, einfach ein neues erzeugen.

  • Passwort ändern
    Durch Klicken dieser Schaltfläche können Sie ein neues Passwort für die Verschlüsselung Ihrer Kontodaten angeben. Das Passwort gilt für alle Konten.
    Wenn Sie ihr Passwort löschen wollen, gehen Sie auf Passwort ändern, lassen Sie das Textfeld für das neue Passwort und die Bestätigung einfach leer und klicken auf OK.

  • Importieren
    Wenn Sie diese Aktion wählen, können Sie ein zuvor gesichertes Konto wiederherstellen.

Informationen zum Kontostand

Hier sehen Sie das Resvolumen, das sich noch auf Ihrem aktuellen Konto befindet. Außerdem wird Ihnen der bisherige Gesamtverbrauch angezeigt und angegeben, welchen Betrag Sie insgesamt auf das Konto einbezahlt haben.

Sie können folgende Aktionen ausführen:

  • Aktivieren
    Mit diesem Knopf aktivieren Sie das gewählte Konto zur Verbindung mit einem Premiumdienst. Das derzeit aktivierte Konto ist hervorgehoben.
    Passwortgeschützte Konten können Sie durch einen Klick auf diesen Knopf und Eingabe Ihres Passworts wieder entsperren.

  • Aktualisieren
    Wenn Sie diese Aktion wählen, wird der aktuelle Kontostand von der Bezahlinstanz abgefragt.

  • Löschen
    Hiermit löschen Sie Ihr Konto.

    ACHTUNG!
    Sollte sich auf Ihrem Konto noch Guthaben befinden, ist dieses unwiederbringlich verloren, wenn Sie keine Sicherheitskopie angelegt haben!

  • Exportieren
    Dieser Knopf speichert Ihre Kontodaten in einer externen Datei. Dies ist sinnvoll, um bei Datenverlust Ihrer Festplatte oder Neuinstallation von JonDo oder des Gesamtsystems wieder Zugriff auf Ihr Konto erhalten zu können. Deshalb ist es die sicherste Lösung, wenn sie Ihr Konto auf ein externes Medium (z.B. einen USB-Stick) speichern.

Erweiterte Einstellungen

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen, wählen anschließend aus der linken Liste den Eintrag Bezahlung und öffnen anschließend den Tab Erweiterte Einstellungen. Diese Einstellungen stehen Ihnen nur in der erweiterten Ansicht zurVerfügung.

In diesem Dialog können Sie erweiterte Einstellungen für die Bezahlung vornehmen.

  • Nicht-anonymisierte Verbindungen zu Bezahlinstanzen erlauben
    Ist diese Option aktiviert, kann JonDo Informationen über Ihren Kontostand abfragen, auch wenn Sie gerade nicht anonym surfen.
    Wird diese Option nicht aktiviert, können nur über eine anonymisierte Verbindung Kontodaten abgerufen werden.
    Es wird deshalb empfohlen, diese Option zu aktivieren.

  • Fehlermeldungen der Abrechnungsinstanz melden (Nur in der beta-Version verfügbar).
    Wenn diese Option aktiviert ist, werden Fehlermeldungen bezüglich der Abrechnungsinstanz angezeigt. Es wird empfohlen, diese Option zu aktivieren.
    In den stabilen Versionen ist dies immer aktiviert.

  • Kontostand automatisch aktualisieren
    Wenn Sie diese Option aktivieren, versucht JonDo in regelmäßigen Zeitabständen Ihre Kontoinformationen zu aktualisieren.
    Wenn Sie dies nicht erlauben, wird in JonDo nicht automatisch der aktuelle Kontostand angezeigt. Das Konto wird bei der Bezahlinstanz dennoch korrekt geführt.

  • Warnen wenn beim Beenden von JonDo nicht von allen Konten eine Kopie existiert
    Sie können JonDo dazu auffordern Sie zu warnen, wenn beim Schließen keine Sicherheitskopie von einem oder mehreren Ihrer Konten vorliegt. Denken Sie daran, dass es keine Möglichkeit gibt Ihr Geld zurückzubekommen, wenn die JonDo-Konfiguration beschädigt oder gelöscht wurde und keine Sicherheitskopie Ihrer Konten bei Ihnen vorliegt.

  • Verbindungs-Timeout
    Setzt das Timeout für Verbindungen zur Bezahlinstanz. Wenn Sie Probleme beim Erzeugen oder Aufladen von Konten haben, kann eine Erhöhung des Timeouts diese in manchen Fällen beheben.
