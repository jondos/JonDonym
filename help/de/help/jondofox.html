JonDoFox

JonDoFox ist ein Firefox-Browser, der speziell für das anonyme Surfen mit JonDonym optimiert ist. Er besteht aus einem portablen Firefox-Browser und einem von JonDos angepassten Profil. Dieses enthält sowohl alle notwendigen Voreinstellungen zum anonymen Surfen über den JonDo, als auch sicherheitskritische Add-ons, die potentielle Schwachpunkte, wie z.B. aktive Inhalte und Cookies absichern. Dabei ist der Browser portabel einsetzbar, d.h. man kann Ihn zusammen mit dem JonDo auf einen Wechseldatenträger kopieren, um ihn auf jedem beliebigen System ohne lokale Installation auszuführen. Hierbei werden keinerlei Spuren auf dem System hinterlassen, auf dem Sie JonDoFox einsetzen. Aus diesem Grund empfiehlt JonDos empfiehlt den Einsatz von JonDoFox, denn dabei entfällt für Sie der Konfigurationsaufwand und die Gefahr, dass Sie beim Surfen Datenspuren hinterlassen, ist minimal.

JonDoFox ist für Windows, Mac OS X und Linux erhältlich. Sie haben auch die Möglichkeit, das JonDoFox-Profil zu Ihrer bereits bestehende Firefox-Installation hinzuzufügen.

Bitte beachten Sie:
Zum anonymen Surfen mit JonDonym genügt es nicht, nur den JonDoFox auszuführen. Hierfür benötigen Sie auch den JonDo-Client, den Sie auf unserer Downloadseite herunterladen können.

Installation unter Windows

Unter diesem Link finden sie ein portables JonDoFox-Paket, das bereits für die Verwendung mit PortableApps vorbereitet ist. Sie können diese unter Windows sowohl auf einem Wechseldatenträger, z.B einem USB-Stick, verwenden oder das Programm lokal auf Ihrer Festplatte ausführen. Kopieren Sie dazu die Datei JonDoFox.paf auf den Datenträger, auf dem Sie JonDoFox ausführen möchten und starten Sie das Programm von dort mit einen Doppelklick auf die soeben kopierte Datei.

Installation unter Mac OS X

Eine portable Version des JonDoFox für Mac OS X können Sie hier erhalten. Diese Version kann sowohl von einem portablen Datenträger als auch lokal von der Festplatte gestartet werden. Zur Installtion doppelklicken Sie bitte auf die dmg-Datei. Kopieren Sie dann einfach die Firefox-Datei auf einen Datenträger Ihrer Wahl. Von dort aus können Sie das Programm einfach durch Doppelklick starten.

Bitte beachten Sie, dass der portable Firefox Mac OS X 10.4 oder neuer erfordert. Für ältere Versionen installieren Sie bitte einen entsprechenden Firefox, und ergänzen Sie Ihn um das JonDoFox-Profil. Eine Anleitung dazu finden Sie hier.

Installation unter Linux

Für Linux gibt es leider derzeit noch keine portable Firefox-Version. Wenn Sie JonDoFox unter Linux verwenden wollen, haben Sie die Möglichkeit eine Firefox-Version zu installieren und diese um das JonDoFox-Profil zu ergänzen. Das Profil ist unter diesem Link erhältlich. (Eine Lite-Version mit weniger Erweiterungen ist ebenfalls erhältlich. Sie bestitzt jedoch alle sicherheitsrelevanten Addons). Zur Installation gehen Sie bitte folgendermaßen vor:

  1. Installieren Sie Firefox, falls dieser noch nicht auf Ihrem System installiert ist. Firefox erhalten Sie unter www.mozilla.com
  2. Entpacken Sie die Datei mit unzip profil.zip.
  3. Starten Sie nun den Firefox mit firefox -P. Dadurch wird der Firefox-Profilmanager geöffnet.
  4. Fahren Sie nun fort, wie unter Hinzufügen des JonDoFox-Profils, mit dem Firefox-Profilmanager beschrieben.

Vorhandende Firefox-Installation um JonDoFox-Profil ergänzen

Besitzen Sie bereits eine Firefox-Installation auf Ihrem System, so können Sie diese um das JonDoFox-Profil ergänzen, um ein Höchstmaß an Anonymitätat zu erreichen, wenn Sie JonDonym nutzen. Jenachdem welches Betriebssystem Sie verwenden, gibt es dazu folgende Vorgehensmöglichkeiten:

  • Windows:
    1. Laden Sie die JonDoFox-Profil-Datei herunter
    2. Bei der Profil-Datei handelt es sich um ein zip-Archiv. Entpacken Sie diese. Dabei erhalten Sie einen Ordner, der profile heißt.
    3. Wählen sie nun: Startmenü -> Ausführen,
    4. Geben sie in das Textfeld ein: firefox -P und drücken Sie auf OK.
      (Hinweis: Achten Sie bitte bei der Angabe von "-P" auf die Groß- und Kleinschreibung).
    5. Fahren Sie fort bei der Beschreibung: Hinzufügen des JonDoFox-Profils, mit dem Firefox -Profilmanager.
  • Mac OS X:
    1. Laden Sie die Image-Datei JonDoFox_OS_X.dmg herunter.
    2. Doppelklicken Sie auf diese Image-Datei. Das Disk-Image öffnet sich daraufhin.
    3. Das Disk-Image enthält die Datei JonDoFox_Install. Doppelklicken Sie auf diese. Dadurch wird eine Installtionsroutine gestartet, die das Profil automatisch zu Ihrer bestehenden Firefox-Installation hinzufügt.
    4. Sollten Sie bereits ein Profil installiert haben, fragt die Installationsroutine, ob Sie das vorhandene Profil überschreiben soll.
    5. Das Installationsprogramm gibt Ihnen eine Rückmeldung über den Erfolg, bevor es sich beendet.

  • Linux:
    Gehen Sie hierzu vor, wie in dieser Anleitung beschrieben.

Hinzufügen des JonDoFox-Profils, mit dem Firefox-Profilmanager

Wenn Sie den Profilmanager geöffnet haben, gehen Sie nun folgendermaßen vor, um das JonDoFox-Profil hinzuzufügen:

  1. Wählen sie nun Profil erstellen.
  2. Es erscheint der Dialog zum Erstellen eines Profils. Klicken Sie auf Fortsetzen.
  3. In dem jetzt erscheinenden Dialog können Sie einen Name für das Profil wählen und den Profil-Ordner angeben. Den Namen für das JonDoFox-Profil können Sie frei wählen. Als Ordner geben Sie bitte das Verzeichnis an, in das Sie die Datei profile.zip entpackt haben.
  4. Beenden Sie die Installation des Profils, indem sie auf Fertigstellen klicken.

Sicherheitskritische Add-ons

Durch eine Reihe wichtiger Firefox-Erweiterungen wird der JonDoFox zum optimalen Werkzeug zum anonymen Surfen. Im JonDoFox-Paket sind bereits alle sicherheitsrelevanten Erweiterungen mit den entsprechende Voreinstellungen vorhanden. Dabei haben Sie die Übersicht über alle wichtigen Erweiterungen in der Firefox-Statusleiste:

So haben Sie stets die Kontrolle über die Funktionen die Ihre Privatspähre gewährleisten. In den folgenden Abschnitten erfahren Sie, wie Sie die Erweiterungen bedienen können, um Ihre Privatsphäre im Web zu maximieren.

Cookies kontrollieren mit CS-Lite

Sicherlich ist Ihre Privatsphäre am besten geschützt, wenn Ihr JonDoFox die Annahme von Cookies vollständig verweigert. Allerdings wird dadurch das Einloggen auf vielen Internetportalen unmöglich. Dank CS-Lite haben Sie die Möglichkeit die Annahme von Cookies so restriktiv wie möglich zu gestalten. Sie können dazu bestimmen, von welchen Betreibern Sie Cookies annehmen möchten und bei welchen Sie dies immer unterbinden wollen. Grundsätzlich erlaubt CS-Lite die Annahme von Cookies nur, wenn Sie eine explizite Freigabe erteilt haben.

Neben der Auswahl, ob Cookies abgelehnt werden sollen oder nicht, können Sie die Freigabe von Cookies weiter einschränken. Wenn Sie mit der linken Maustaste auf das CS-Lite Icon . Darufhin erscheint folgendes Menü:

Hier haben Sie die Möglichkeit, folgende Berechtigungen zu gewähren:

  • Temporäre Berechtigung:
    In diesem Fall wird beim nächsten Start des JonDoFox kein Cookie mehr von dieser Seite akzeptiert, bzw. wenn Sie dies wünschen müssen Sie erneut ein entsprechende Berechtigung dazu geben. Außerdem werden die Cookies beim Beenden von JonDoFox gelöscht.

  • Berechtigen während Sitzung:
    Cookies werden beim nächsten Start von JonDoFox wieder von dieser Seite akzeptiert. Der Cookie wird beim Beenden von JonDoFox gelöscht.

  • Uneingeschränkte Berechtigung:
    Cookies werden beim nächsten Start von JonDoFox wieder von dieser Seite akzeptiert. Nur abgelaufene Sitzungs-Cookies von dieser Seite werden beim Beenden von JonDoFox gelöscht.

Wenn Sie die Cookies sofert entfernen möchten, klicken Sie auf Cookies entfernen. Sie haben hier auch die Möglichkeit, bereits erteilte Berechtigungen wieder zu entziehen, indem Sie den Eintrag Blockieren wählen. Ebenfalls in diesem Menü finden Sie den Eintrag Cookies global akzeptieren, um Cookies grundsätzlich von allen Seitenbetreibern anzunehmen.

Hinweis:
Wenn Sie Cookies global akzeptieren, werden grundsätzlich alle Cookies vom Browser angenommen. Sie erhalten keine Rückmeldung, ob Cookies angenommen werden sollen und wenn ja, von wem. Dadurch ist Ihre Privatspäre nicht mehr gewährleistet.

Schaden durch aktive Inhalte vermeiden mit NoScript

NoScript ist eine Erweiterung, mit der kontrolliert werden kann, ob aktive Inhalte beim Besuch einer Seite ausgeführt werden dürfen oder nicht. Primär können Sie durch NoScript bestimmen, ob und wenn ja welchen Betreiber Sie die Berechtigung erteilen möchten JavaScript-Funktionen auszuführen. NoScript ist eine äußerst sicherheitskritische Erweiterung, den JavaScript-Funktionen können nicht nur schadhafte Funktionen auf Ihrem System ausführen sondern auch identifizierende Eigenschaften von Ihrem Browser auslesen, wie z.B. welche Browser Sie verwenden, auf welchem Betriebssystem er läuft, welche Größe Ihr Browserfenster hat, uvm. Besuchen Sie den JonDos-Anonymitätstest, um zu erfahren, welche Informationen über Ihren Computer sich mit JavaScript ermitteln lassen.

Mithilfe von NoScript können Sie nicht nur JavaScript-Funktionen an der Ausführung hindern, sondern auch vermeiden, dass andere aktive Inhalte wie Java oder Flash Schaden anrichten können. Dies ist möglich, da Sie mit NoScript, die entsprechenden Plugins deaktivieren, die die entsprechenden Inhalte ausführen.

Kontrolle von JavaScript-Funktionen:
In den JonDoFox-Voreinstellungen werden zunächst alle JavaScript-Inhalte blockiert, wenn Sie eine Seite besuchen. Klicken Sie mit der linken Maustaste auf das NoScript-Icon in der Statusleiste. Sie erhalten nun eine Auflistung, welche Seitenbetreiber JavaScript-Funktionen ausführen wollen. Die entsprechenden Einträge sind durch das Icon gekennzeichnet:

Erteilen Sie die Erlaubnis durch ein Klick auf temporär erlauben bei dem entsprechenden Listeneintrag. Auf die gleiche Weise können Sie eine bereits erteilte Berechtigung wieder entziehen, wie in der rechten Grafik veranschaulicht ist. Wenn Sie allen Betreiber die bereits erteilten temporären Berechtigungen sofort entziehen wollen, können Sie dies, indem Sie auf den Eintrag Temporäre Berechtigungen aufheben klicken.

Hinweis:
Wenn Sie eine Berechtigung erteilen oer gewähren, wird die Seite mit den jeweiligen Berechtigungen neu geladen.

Sollten Sie einem Betreiber grundsätzlich nicht vertrauen, können Sie ihn als nicht vertrauenswürdig einstufen. Dies bewirkt, dass Javascript-Funktionen von diesem Anbieter auch keine temporäre Berechtigung gewährt werden kann. Um diese Einstufung durchzuführen, fahren Sie mit der Maus über Nicht vertrauenswürdig. Sie erhalten dann die Übersicht der entsprechenden Anbietern, denen Sie das Vertrauen entziehen können:

Die Aktion kann auf die gleiche Weise wieder rückgängig gemacht werden.

Kontrolle der installierten Plugins:
Wenn Sie eine Webseite besuchen, die aktive Inhalte, wie z.B. Flash-Movies oder Java-Applets über ein Plugin laden will, wird dies von NoScript durch die entsprechenden JonDoFox-Voreinstellungen verhindert. Anstelle des aktiven Inhalts erscheint ein NoScript-Platzhalter (linkes Bild):

Mit einem Klick auf diesen erscheint ein Dialog, durch den Sie den Inhalt freigeben können. Wenn Sie die Seite erneut laden, wird der aktive Inhalt von NoScript wieder blockiert. Eine temporäre Freigabe von aktiven Inhalten durch Plugins können Sie auch gewähren. Indem Sie mit der Maus über den Punkt Geblockte Objekte fahren. Es erscheint eine Liste, wie sie das rechte Bild zeigt. In dieser erhalten Sie die Übersicht über alle blockierten aktiven Inhalte einschließlich der zugehörigen Plugins. Auch blockierte JavaScript-Inhalte sind in dieser Auflistung enthalten. Klicken sie auf einen der Einträge können sie damit die Ausführung des entsprechenden aktiven Inahlts temporär erlauben, bzw. wieder verbieten.

Einfaches Umschalten mit SwitchProxy

Wenn Sie ohne JonDo im Web surfen wollen, müssen Sie dazu nicht einen neuen Browser straten oder die Proxyeintsellungen von JonDoFox mühsam umkonfigurieren. Verwenden Sie einfach die SwitchProxy-Erweiterung, um bequem zwischen JonDo und direkter Internetverbindung hin- und herzuschalten. Wenn Sie mit der rechten Maustaste auf die SwitchProxy-Schaltfläche in der Statusleiste klicken, haben Sie bei JonDoFox bereits zwei Auswahlmöglichkeiten:

  • None: In diesem Fall nimmt JonDoFox direkte Verbindung zum Internet auf und Sie können surfen auch wenn JonDo nicht gestartet ist.
  • JonDo: Dies entspricht der Standardeinstellung. JonDoFox schickt hier alle Daten über den JonDo.

Bitte beachten Sie:
Wenn Sie den Eintrag None ausgewählt haben, wird Ihre tatsächliche IP-Adresse sichtbar für die Betreiber, deren Seiten Sie aufrufen. Ihre Privatspähre kann dadurch nicht mehr sichergestellt werden.
