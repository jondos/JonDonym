Services: Mixminion

This panel is only available in Expert View and only in the beta version. To get to this settings window press the button Settings in the main window of JonDo, then choose the entry Services and open the tab Mixminion.

On this page you can find information and settings for the Mixminion anonymization system. Mixminion is a software for anonymously sending and receiving emails. Just as with Tor, we can not guarantee for completeness or functionality of the option as it is experimental. Further information about Mixminion can be found at www.mixminion.net.

If you would like to use JonDo to access the Mixminion system you need to enter it as a proxy on port 25 in your email client.

  • Activate
    Using Mixminion with JonDo is still experimental. Deactivating this checkbox will lower memory usage by JonDo and prevent accidentally using Mixminion.

  • Available Mixminion Nodes
    This tab shows systems that offer a connection to the Mixminion network.

  • Reload
    This button retrieves an updated list of currently accessible Mixminion nodes.

  • Preferences for anonymous routing

    • Nodes per connection
      Here you can set the number of Mixminion nodes through which a connection should be routed before being sent to the destination system. The more nodes you use, the more effective the anonymization of the connection will be.

  • Email settings for reply mails

    • My email address
      Enter here your email address used for reply mails. If you do not specify an address JonDo will use the address from your email client.
