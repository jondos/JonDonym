Java-Software (JRE) aktualisieren

Hinweis: Nutzer von MacOS X beziehen Updates von Java automatisch über das Betriebssystem und müssen keine manuelle Aktualisierung vornehmen. Die folgenden Ausführung sind für sie nicht relevant.

JonDo ist in der Programmiersprache Java geschrieben und benötigt deshalb ein installiertes Java zur Ausführung. Bitte beachten Sie , dass JAP/JonDo momentan nur mit den Java-Distributionen Sun-Java und OpenJDK kompatibel ist.

Sofern Sie einen anderen Browser als JonDoFox verwenden, wird Java bei der Installation in Ihrem Browser meist als Plugin eingerichtet. Dieses ist für JonDo nicht notwendig, führt allerdings auch dazu, dass Ihr Browser nun Java-Webanwendungen (Applets) ausführen kann, die auch Ihre Anonymität aufheben können. Es wird deshalb sehr empfohlen, dieses Plugin in Ihrem Browser zu deaktivieren oder zu filtern. Ferner werden auch in diesem Plugin, wie in jeder Software, immer wieder Fehler gefunden von denen einige auch die Sicherheit des Computers, auf dem diese installiert ist, gefährden können. JonDo weist Sie deshalb darauf hin wenn Updates für Ihr installiertes Java vorliegen, und wir empfehlen dringend die Aktualisierung jeweils sofort durchzuführen.

Obwohl Java mehrere gleichzeitige Installationen verschiedenen Versionen unterstützt, sollten Sie möglichst immer nur EINE EINZIGE Java-Version auf Ihrem Rechner installiert haben. Deinstallieren Sie deshalb vor einer neuen Java-Installation alle alten Versionen dieser Software.

Hinweis: Die folgende Anleitung bezieht sich auf die Installation unter Windows. Bei Linux ist die Installation distributionsspezifisch. Eventuell steht nicht für jede Linux-Distribution die neueste Java-Version zur Verfügung.

Die Aktualisierung ihrer Java-Version in drei Schritten:

  1. Herunterladen der neuen Version
    Laden Sie sich die aktuelle Version von dieser Seite herunter (alternativ von dieser Seite für ältere Windows-Versionen; bitte beachten Sie, dass Sun-Java-Version ab 1.4 nicht auf Windows 95 laufen).

  2. Deinstallieren der alten Version
    Öffnen Sie im Startmenü die "Systemsteuerung", dort wählen Sie "Software". Es kann eine Weile dauern, bis die Liste aufgebaut wurde. Wählen Sie in der Liste den Eintrag "J2SE Runtime Environment" und wählen Sie "Entfernen". Warten Sie, bis die Deinstallation abgeschlossen ist. Eventuell müssen Sie den Rechner neu starten.

  3. Installieren der neuen Version
    Doppelklicken Sie auf das Installationsprogramm, das Sie im Schritt 1) heruntergeladen hatten. Folgen Sie den Installationsanweisungen. Die Installation kann einige Zeit in Anspruch nehmen. Eventuell müssen Sie den Rechner danach neu starten.
