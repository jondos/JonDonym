Vermeiden von Datenspuren im Internet: PrefBar

PrefBar so konfigurieren, dass die Anonymität erhöht wird

Webseiten können Inhalte anbieten, die Ihre Identität aufdecken können, obwohl sie mit JonDo anonym surfen. Das kann z.B. über JavaScript oder auch Java geschehen.
Um diese Inhalte zu blocken, kann PrefBar eingesetzt werden. PrefBar ist ein Plugin für Firefox und Mozilla Seamonkey.

Installation von PrefBar

  • Öffnen Sie ihren Browser

  • Öffnen Sie die Homepage von PrefBar

    Gehen Sie dazu auf diese Seite.

  • Installieren Sie PrefBar

    Klicken Sie auf den Link für einen der beiden Installer. Dabei wird möglicherweise ein Popup oder ein ähnlicher Hinweis angezeigt, der entweder fragt ob die Installation zugelassen werden darf oder nur aussagt dass die Installation geblockt wurde. In jedem Fall können Sie dabei einstellen, dass die Installation fortgesetzt werden soll. In Firefox müssen sie eventuell Installationen für diese Seite explizit zulassen.
    Nach der Installation sollten Sie ihren Browser neu starten.

Einstellungen für PrefBar
  • Öffnen Sie ihren Browser

    Sie sollten jetzt eine neue Toolbar sehen:

  • Einstellungen vornehmen

    Klicken Sie auf Anpassen. Daraufhin sehen Sie folgendes Fenster:

    Wählen Sie aus den verfügbaren Elementen das Element Java und klicken Sie auf Hinzufügen ->. Wiederholen Sie diesen Schritt für das Element Referrer senden. Nach einem Klick auf OK sehen Sie in der Toolbar noch eine Checkbox für Java und Referrer senden.
    Sie können natürlich auch noch andere Elemente hinzufügen bzw. entfernen, je nachdem was Sie für nützlich halten.

  • Inhalte blockieren

    Für die Checkboxen gilt: Wenn sie angehakt sind, wird dieser Inhalt nicht blockiert. Wollen Sie z.B. Javascript blockieren, müssen Sie Javascript abhaken.
    Es wird dringend empfohlen, Java, JavaScript und Referrer senden zu deaktivieren.

Hinweis: PrefBar können Sie mit F8 anzeigen oder verstecken.

Test: Unterziehen Sie diese Einstellungen einem Anonymitätstest.
