Forwarding-Server

Dieses Panel steht Ihnen in der erweiterten Ansicht zur Verfügung. Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Forwarding-Server.

Ein Forwarding-Server dient zur Umgehung von Zugangssperren zu JonDonym-Diensten. Durch Zensur oder andere restriktive Maßnahmen ist es manchen JonDo-Nutzern nicht möglich, eine direkte Verbindung zu JonDonym-Diensten herzustellen. Aus diesem Grund kann ein JonDo, dem dies möglich ist, als Forwarding-Server agieren, über den andere JonDo-Nutzern sich mit Mixkaskaden verbinden können. Wenn Sie Ihren JonDo als Forwarding-Server freischalten möchten, nehmen Sie in diesem Fenster die entsprechenden Einstellungen vor.

Bitte beachten Sie:
Wenn Sie den Forwarding-Server aktivieren, kann jeder JonDonym-Nutzer Ihren JonDo als Zugangspunkt verwenden. Sie gehen auch das Risiko ein, dass Nutzer Ihren JonDo verwenden können, um strafbare Handlungen im Internet durchzuführen.

Forwarding-Server Einstellungen

  • Portnummer:
    Tragen Sie hier den Port ein, auf dem Ihr JonDo Forwarding-Verbindungen annimmt. Dazu ist es erforderlich, dass Sie diesen Port in Ihrer Firewall für eingehende Verbindungen freischalten.

  • Meine Internetverbindung:
    Wählen Sie hier die Art und Geschwindigkeit Ihrer Internetverbindung aus. Dadurch wird der Wert im Feld "Max. Bandbreite" automatisch gesetzt.

  • Anteil für Forwarding:
    Wählen Sie in diesem Feld den Prozentsatz aus, der von Ihrer gesamten Upload-Bandbreite für das Forwarding zur Verfügung stehen soll.

  • Max. benutzte Bandbreite:
    Hier wird Ihnen angezeigt, wie viel Bandbreite tatsächlich gemäß den obigen Einstellungen für das Forwarding genutzt wird.

Erlaubte Mixkaskaden

In diesem Abschnitt haben Sie grundsätzlich zwei Einstellungsmöglichkeiten. Entweder, Sie erlauben allen Clients, die Ihren JonDo als Forwarder benutzen, den Zugriff auf alle Ihrem JonDo bekannten Kaskaden ("Erlaube Client-Zugriff auf alle verfügbaren Mixkaskaden" ist markiert) oder Sie erlauben den Zugriff nur auf bestimmte Kaskaden. Auf welche Kaskaden zugegriffen werden darf, legen Sie mit den Schaltflächen "Zu erlaubten Kaskaden hinzufügen" und "Von erlaubten Kaskaden entfernen" fest. Der Zugriff ist dann nur auf Kaskaden möglich, die sich im Feld "Erlaubte Mixkaskaden" befinden.

InfoService Anmeldung

Damit JonDo-Benutzer, die einen Forwarding-Server benötigen, Ihren JonDo finden können, muss Ihr JonDo bei mindestens einem InfoService als Forwarder registriert sein. Sie können sich bei allen InfoServices oder nur bei bestimmten anmelden. Die Verfahrensweise ist analog zum Vorgehen bei den erlaubten Mixkaskaden.
