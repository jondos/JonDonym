JonDo User Interface: Iconified window

JonDo's iconified window is a minimalistic view of JonDo's main window. You can invoce it by pressing or by double-clicking anywhere into the main window. It shows only the most important information and consists of the following elements:

  • Bytes This shows the amount of traffic that has been anonymized during this settion
  • Anon Shows the level of anonymity(low, fair, high)
  • for toggling anonymity on or off
  • button wich switches back to normal view (double-clicking the window will also do)
Right-clicking on the window will open a menu which enables you, besides other actions, to view the currently used service filter (boldface) and cascade, and to choose a new one.
