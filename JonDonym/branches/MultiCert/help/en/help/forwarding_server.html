Forwarding server

This panel is only available in Extended View. To get to this panel, press the button Settings in JonDo's main window, then choose the entry Forwarding Server from the list to the left.

A forwarding server is used to bypass any blocking of access to JonDo services. Because of censorship or other restrictive measures, a user may not be allowed to connect directly to the anonymization service. This is why JonDo includes the possibility of forwarding connections to the services. If you would like your JonDo to also forward connections, you can find the relevant settings in this window.

The option Forwarding Server ON activates the forwarding server.

Beware
After activating forwarding, every JonDonym user can use your JonDo as an access point to the services. You are exposing yourself to the risk of users committing crimes on the Internet through your JonDo; however, you are still well protected from discovery of your own IP address by the anonymization services.

Forwarding server settings

  • Server port for forwarding
    Enter the port number that your JonDo program uses for incoming forwarding connections. This port must be open for incoming connections in your firewall.

  • My Internet connection
    Select the type and speed of your internet connection. The value in the field Max. bandwidth is automatically set by this.

  • Percentage for forwarding
    Choose a percentage from this field to set how much of your total upstream bandwidth should be used for forwarding.

  • Max. used bandwidth
    The bandwidth actually used for forwarding based on the settings above is shown here.

Allowed services

This section allows for basically two settings. Either you allow all forwarded clients to access all JonDo services known (Allow client access to all known services is checked), or you only allow access to certain services. Service access is set with the buttons Add to allowed services and Remove from allowed services. Access is allowed only to services found in the Allowed Services list.

InfoService registration

For JonDo users to find your JonDo forwarding server, your JonDo must be registered as a forwarder with at least one InfoService. You can register with all InfoServices or only certain ones. The procedure is basically the same as above.
