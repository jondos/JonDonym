Dienste: Erweiterte Einstellungen

Dieses Panel steht Ihnen in der erweiterten Ansicht zur Verfügung. Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen, wählen anschließend aus der linken Liste den Eintrag Dienste und öffnen anschließend den Tab Erweiterte Einstellungen.

Folgende Allgemeine Einstellungen, die alle Dienste betreffen, können Sie hier vornehmen:

  • Anonymisiere HTTP-Header
    Wenn diese Option aktiviert ist, setzt der JonDo die vom Browser übertragenen HTTP-Header auf Standardwerte, die auch von JonDoFox gesetzt werden. Dadurch sind Nutzer von anderen Browsern nicht unterscheidbar von JonDoFox-Nutzern oder solchen, die die HTTP-Filterung ebenfalls aktiviert haben. Dies verbessert den Schutz der Privatsphäre.

    Hinweis
    Diese Funktion steht nur für unverschlüsselte Verbindungen zur Verfügung. Eine entsprechende Filterung bei verschlüsselten Verbindungen kann von JonDoFox vorgenommen werden.

  • Jede Webseite einzeln bestätigen wenn Anonymitätsmodus aus
    Ist dieses Feld markiert, fragt JonDo bei deaktiviertem Anonymitätsmodus für jede angeforderte Webseite nach einer Bestätigung, um zu vermeiden, dass unbemerkt Seiten ohne Schutz aufgerufen werden können. Wenn diese Option deaktiviert ist, kann der JonDo sich auch im nicht-anonymen Modus direkt mit einem Webserver verbinden, ohne ständig eine Bestätigung vom Nutzer zu fordern. Eine einmalige Bestätigung ist aber weiterhin erforderlich.
    Weitere Informationen zum nicht-anonymen Weiterleiten von Daten

  • Anonymitätsservice sofort nach Programmstart aktivieren
    Ist dieses Feld markiert, so baut der JonDo gleich nach dem Start eine Verbindung zum Anonymisierungsdienst auf.

  • Automatisch wieder verbinden nach Verbindungsabbruch
    Setzen Sie hier einen Haken, wenn Sie wollen, dass der JonDo automatisch die Verbindung zum Anonymiserungsdienst wieder herstellt, wenn diese unterbrochen wurde.

  • Mix-Kaskaden automatisch wechseln
    Bei dieser Option wird beim Start von JonDo eine zufällige Kaskade aus den durch Ihren Dienstfilter festgelegten Kaskaden ausgewählt. Während des Betriebs wird die Kaskade nur dann automatisch gewechselt, wenn die aktive Kaskade ausfällt.

  • auch beim Start von JAP/JonDo
    Wird diese Option deaktiviert, verbindet sich JonDo beim Start mit derselben Kaskade mit der er vor dem letzten Beenden verbunden war.

  • Nur mit explizit freigeschalteten Kaskaden verbinden (Whitelist)
    Bei Aktivierung wird JonDo nur Verbindungen zu Kaskaden erlauben, die Sie explizit überprüft und freigeschaltet haben. Dadurch sind Sie optimal davor geschützt, sich versehentlich mit nicht-vertrauenswürdigen JonDonym-Diensten zu verbinden. Allerdings profitieren Sie nicht automatisch davon, wenn neue Dienste verfügbar sind, sondern müssen diese jeweils einzeln in der JonDonym-Dienst-Ansicht freischalten.

  • Sende Dummy-Daten
    Hier können Sie festlegen, wie häufig Pakete ohne Nutzinhalt verschickt werden, um für einen Beobachter eine laufende Kommunikation vorzutäuschen.

  • Anmelde-Timeout
    Sollte Ihre Internetverbindung zu langsam sein, so kann ein zu geringes Timeout beim Anmelden an den Dienst dazu führen, dass JonDo keine anonyme Verbindung aufbauen kann. Ein zu hohes Timeout führt unter Umständen zu hohen Wartezeiten beim Verbinden, wenn der aktuell gewählte Dienst zu langsam oder nicht verfügbar ist. Stellen Sie das Timeout deshalb je nach Bedarf höher oder niedriger ein.
