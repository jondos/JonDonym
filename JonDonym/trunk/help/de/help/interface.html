Die JonDo Oberfläche

Anonymität aktivieren
Um anonymes Surfen möglich zu machen muss als erstes der JonDo gestartet werden. Danach müssen Sie das Kästchen "Ein" unter "Anonymität" aktivieren. Wenn der Verbindungsaufbau erfolgreich war erscheint kurz danach ein Zeiger im Anonym-O-Meter. Außerdem wird die Nutzerzahl des aktuell gewählten Anonymisierungsdienstes angezeigt.

Hinweis auf notwendige Aktualisierung
Oben rechts im Fenster neben dem JonDo-Logo finden Sie einen Hinweis, der Sie zum Aktualisieren der JonDo-Software bzw. der Java-Version auffordert. Ein Klick darauf zeigt ihnen, was Sie aktualisieren sollten. Sollte JonDo eine Aktualisierung benötigen, öffnet sich dabei das Updatefenster. Im Falle von Java öffnet sich dabei ein Hinweis, daß eine neue Java-Version vorhanden ist. Ein Klick auf "Hilfe" in diesem Popup erklärt Ihnen, wie sie Java aktualisieren können. Sehen Sie keinen solchen Hinweis, sind sowohl Ihr JonDo als auch Java auf dem neuesten Stand.
Rechts daneben sehen Sie die aktuelle Versionsnummer. Ein Klick darauf zeigt Ihnen Informationen zu der Version und den Entwicklern.

Dienstauswahl
Direkt unter dem Logo befindet sich die Auswahlliste der verfügbaren Dienstfilter. Wählen Sie eine Mixkaskade innerhalb eines Filters aus, verbindet sich JonDo mit dieser Mixkaskade, und sperrt automatisch die Verbindung zu allen Mixkaskaden außerhalb dieses Filters (der Filter wird aktiviert).

Das Symbol daneben mit den runden Pfeilen aktualisiert die Liste. Die Schaltfläche "Details" bringt Sie in das Einstellungsfenster von JonDo, wo sie Genaueres zu den verfügbaren Diensten erfahren können.

Anonym-O-Meter
Das Anonym-O-Meter informiert den Nutzer über die aktuelle Auslastung des Anonymisierungsdienstes. Im einzelnen sind dies:

  • Die Anzahl der momentan aktiven Nutzer, die mit dem Internet verbunden sind und im JonDo den anonymen Zugriff aktiviert haben.
  • Die Verkehrssituation, d.h. die Menge der Daten, die von dem Anonymisierungsdienst pro Zeiteinheit verarbeitet und weitergeleitet werden.
Aus diesen Informationen kann ein teilweise subjektives Beobachtungsrisiko und eine erreichbare Anonymität abgeleitet werden. Je mehr Nutzer aktiv sind und auch Nachrichten senden bzw. empfangen, um so größer ist die Anonymitätsgruppe jeder einzelnen ins Internet gesendeten Nachricht, da potentiell jeder Nutzer die Nachricht gesendet haben könnte. Nimmt man an, dass der Angreifer eine Anzahl Nutzer kontrolliert bzw. selbst angebliche Nutzer simuliert, so fällt es ihm mit größer werdender Anonyitätsgruppe um so schwerer, alle bis auf einen Nutzer zu kontrollieren. In späteren Versionen soll diese Art Angriff dadurch verhindert werden, dass die Nutzer sich bei den Mixen mit Hilfe sogenannter Tickets, d.h. auf ein Pseudonym ausgestellte digitale Berechtigungsmarken, die mit Hilfe sog. Blinder Digitaler Signaturen gebildet werden, anmelden.

Klicken Sie hier, um eine Beschreibung für die Zustände des Anonym-O-Meters zu bekommen.

Kontostand
Premiumdienste buchen den Traffic von ihrem Konto ab. Hier können Sie sehen, welchen Kontostand Sie momentan haben.
Um die erweiterte Kontenansicht zu öffnen, klicken sie auf den Pfeil links neben der Restvolumensanzeige. Hierdurch erhalten sie weitere Informationen zu ihrem Konto:

  • wieviel von Ihrem Konto Sie in dieser Sitzung verbraucht haben,
  • wieviel von Ihrem Konto Sie insgesamt verbraucht haben und
  • wann Ihre Kontoinformationen das letzte Mal aktualisiert wurden.

Klicken sie wieder auf den Pfeil, um die erweiterte Ansicht zu schließen.

Eigene anonymisierte Daten
Zur Information des Nutzers und zur Überprüfung der Funktionalität wird von JonDo angezeigt, wie viele Daten der Nutzer in der aktuellen Sitzung bereits über den Anonymitätsdienst übertragen hat. Mit dem Pfeil links neben der Datenanzeige können Sie die erweiterte Ansicht öffnen. Hier sind die anonymisierten Daten weiter unterteilt in:

  • World Wide Web
  • anderen Internetdienste

Die erweiterte Ansicht läßt sich durch erneutes Klicken auf den Pfeil schließen.

Forwarder

Durch die Forwarding-Funktion können andere JonDo-Benutzer über ihr System eine Verbindung zu einem Anonymisierungsdienst aufbauen. Dies ist hilfreich, wenn der Internetprovider oder eine staatliche Hoheit eines JonDo-Benutzers den Zugang zum Anonymisierungsdienst unterbindet.

Achtung :
Dadurch besteht für Sie die Gefahr in den Verdacht einer Straftat zu geraten, die von einem JonDo-Benutzer begangen wurde, der ihr System als Forwarder nutzt.
Wenn Sie auf den Pfeil links neben der Anzeige klicken, erhalten Sie folgende Informationen zum Forwarder:

  • Wieviele weitergeleitete Verbindungen
    • momentan gehalten werden
    • angenommen wurden
    • abgelehnt wurden
  • Wieviel weitergeleitetes Datenvolumen bereits angefallen ist
  • Welche Bandbreite dafür verwendet wird.

Weitere Schaltflächen

  • Die Schaltfläche "Minimieren" (Symbol: ) lässt JonDo auf ein Fenster schrumpfen, das nur noch die wichtigsten Informationen anzeigt. Dieses Fenster dockt automatisch am Bildschirmrand an. Dort können Sie durch einen Klick auf das JonDo-Icon die Anonymität ein- oder ausschalten. Alternativ können Sie das Minifenster auch aufrufen, wenn Sie im JonDo Hauptfenster doppelklicken.
  • Hilfe öffnet die Hilfe zu JonDo
  • Assistent hilft Ihnen beim Einrichten von JonDo
  • Einstellungen öffnet das Fenster, in dem Sie alle Einstellungen für JonDo vornehmen können
  • Beenden trennt die Verbindung zu den Diensten und beendet JonDo
