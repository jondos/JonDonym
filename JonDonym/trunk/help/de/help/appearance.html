Verändern des Erscheinungsbildes

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Erscheinungsbild.
In diesem Dialog können Sie Einstellungen vornehmen, die die grafische Erscheinung, die Sprache und das Anzeigeverhalten nach dem Starten von JonDo festlegen.

Erscheinungsbild

  • Erscheinungsbild:
    JonDo unterstützt verschiedene Formen der Darstellung von Fenstern und Bedienelementen.
    • Sprache
      Hier können Sie die Sprache einstellen, in welcher die Bedienelemente erscheinen sollen.
      Auch hier ist ein sofortiger Neustart von JonDo erforderlich um die Änderung zu übernehmen.

    • LookAndFeel
      Wählen Sie hier ihr bevorzugtes LookAndFeel aus der Liste aus. Die Änderungen erfordern einen sofortigen Neustart von JonDo.
      Sie können neue LookAndFeel-Optionen hinzufügen, indem Sie sich aus dem Internet neue Pakete herunterladen und in JonDo importieren. Diese sind nicht JonDo-spezifisch, Sie können alle Pakete für Java (Swing) verwenden.
      Solche Pakete finden Sie z.B. auf www.javootoo.com. Laden Sie ein solches Paket auf ihren Computer und klicken Sie auf importieren um die neue Option hinzuzufügen.
      • Entfernen
        Damit können Sie ein LookAndFeel löschen, wobei dies nur bei solchen funktioniert, die sie selbst herunter geladen haben. Das aktuell ausgewählte LookAndFeel können Sie ebenfalls nicht löschen. Das LookAndFeel bleibt dabei bis zu einem Neustart von JonDo im Speicher.

      • Importieren
        Ein Klick darauf öffnet ein Auswahlfenster. Navigieren Sie zum Speicherort des neuen LookAndFeel Pakets und wählen Sie die Datei mit der Endung ".jar".

    • Schriftgrösse
      Hier können Sie die Schriftgrösse der Bedienelemente einstellen.
      Diese Änderungen werden sofort aktiv.

Ansicht von JonDo

  • Erweiterte Ansicht:
    In der erweiterten Ansicht werden alle Optionen im Hauptfenster sowie in den Konfigurationsdialogen angezeigt.

  • Vereinfachte Ansicht:
    In der vereinfachten Ansicht werden einige fortgeschrittene Optionen nicht angezeigt. Die Grundfunktionalität des JonDo ändert sich dadurch nicht.

  • Mini-Ansicht immer im Vordergrund
    Schiebt die Mini-Ansicht von JonDo immer vor alle anderen Fenster des Betriebssystems. Wenn die Option abgewählt ist, wird die Mini-Ansicht hinter anderen Fenstern versteckt.
Diese Änderungen erfordern einen sofortigen Neustart von JonDo.

Programmstart

  • Gehe nach dem Start...:
    Wenn Sie diese Option aktivieren, haben Sie zwei weitere Auswahlmöglichkeiten:

    • ...in die Minimalansicht:
      Diese Option minimiert das Hauptfenster des JonDo sofort nach dem Start.

    • ...in den Systray:
      Wenn diese Einstellung gewählt wurde, verschwindet das Hauptfenster des JonDo gleich nach dem Start im Systray von Windows.

Fensterposition merken

Diese Option steht nur in der erweiterten Ansicht zur Verfügung.

Wenn Sie bei einem der folgenden Kästchen einen Haken setzen, dann erscheint das entsprechende Fenster von JonDo bei jedem Start an der gleichen Stelle, an der es sich beim vorherigen Beenden befand.

  • Hauptfenster
  • Einstellungsfenster
  • Mini-Fenster
  • Hilfefenster

Fenstergröße merken

Diese Option steht nur in der erweiterten Ansicht zur Verfügung.

  • Hilfefenster
    Wenn Sie diesen Haken aktivieren, wird sich JonDo die Größe des Hilfefensters merken.

Programmende

Diese Option steht nur in der erweiterten Ansicht zur Verfügung.

  • Warnung beim Beenden von JonDo
    Diese Option zeigt eine Warnung beim Beenden von JonDo um darauf aufmerksam zu machen, dass die Proxykonfiguration im Browser geändert werden muss, wenn man weitersurfen möchte.
