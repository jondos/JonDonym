Vermeiden von Datenspuren im Internet: Mozilla Firefox

Öffnen Sie im Menü Extras den Punkt Einstellungen.

  • Cookies:
    Wählen Sie den Tab Datenschutz und haken Sie Cookies akzeptieren an. Wählen Sie bei Cookies behalten: die Option bis Firefox geschlossen wird. Dadurch können Sie alle Seiten besuchen, die zwingend Cookies voraussetzen, können jedoch durch Cookies nicht wiedererkannt werden.

  • Java & JavaScript:

    Wählen Sie den Tab Inhalt. Deaktivieren die Häkchen bei Java aktivieren und JavaScript aktivieren.

Hinweis:
Das sind lediglich grundlegende Einstellungen. Die Verwendung von JonDoFox verhindert das Senden sämtlicher digitaler Spuren und die Ausführung von schadhaften aktiven Inhalten.

Empfohlene Extensions

  • CS Lite
    Diese Extension ermöglicht eine flexiblere Verwaltung Ihrer Cookies und verhindert, dass Sie Cookies von Webseiten erhalten, denen Sie nicht vertrauen.

  • NoScript
    Blockiert grundsätzlich JavaScript und andere aktive Inhalte. Durch eine explizite Freigaben können Sie jedoch selbst entscheiden, welchen Webseiten sie das Senden und Ausführen aktiver Inhalte in Ihrem Browser erlauben.

Diese beiden Extension sind im JonDoFox bereits vorinstalliert.

Test:
Unterziehen Sie diese Einstellungen dem JonDos-Anonymitätstest.
