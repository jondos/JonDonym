Java-Software (JRE) aktualisieren

JonDo ist in der Programmiersprache Java geschrieben und benötigt deshalb ein installiertes Java zur Ausführung. Die Installation von Java führt allerdings auch dazu, dass ein sogenanntes Plugin in den Browser integriert wird, mit dem dieser nun Java-Webanwendungen (Applets) ausführen kann. Leider werden auch in diesem Plugin, wie in jeder Software, immer wieder Fehler gefunden, von denen einige auch die Sicherheit des Computers, auf dem diese installiert ist, gefährden können. JonDo weist Sie deshalb darauf hin wenn Updates für Ihr installiertes Java vorliegen, und wir empfehlen dringend die Aktualisierung jeweils sofort durchzuführen. Außerdem sollten Sie das Plugin in Ihrem Browser immer deaktivieren, da dessen Ausführung trotz aktivem JonDo zur Deanonymisierung Ihrer Internetverbindung führen kann.

Obwohl Java mehrere gleichzeitige Installationen verschiedenen Versionen unterstützt, sollten Sie außerdem immer nur EINE Java-Version auf dem Rechner installiert haben. Deinstallieren Sie deshalb vor einer neuen Java-Installation alle alten Versionen dieser Software.

Das Aktualisieren ihrer Java Version unter Windows erfolgt in drei Schritten:

  1. Herunterladen der neuen Version
    Laden Sie sich die aktuelle Version von dieser Seite herunter (bitte beachten Sie, dass Sun-Java-Version ab 1.4 nicht auf Windows 95 laufen). Sie benötigen die aktuelle Version des "Java Runtime Environment (JRE) <Versionsnummer>". Klicken Sie auf "Download". Klicken Sie in der folgenden Seite auf "Accept License Agreement". Klicken Sie anschließend auf den Link "Windows Offline Installation, Multi-language". (Auch die "Online Installation" ist möglich, dann müssen Sie zum Zeitpunkt der Installation allerdings online sein, da der Installer Daten aus dem Internet nachladen muss).

  2. Deinstallieren der alten Version
    Öffnen Sie im Startmenü die "Systemsteuerung", dort wählen Sie "Software". Es kann eine Weile dauern, bis die Liste aufgebaut wurde. Wählen Sie in der Liste den Eintrag "J2SE Runtime Environment <Versionsnummer>" und wählen Sie "Entfernen". Warten Sie, bis die Deinstallation abgeschlossen ist. Eventuell müssen Sie den Rechner neu starten.

  3. Installieren der neuen Version
    Doppelklicken Sie auf das Installationsprogramm, das Sie im Schritt 1) heruntergeladen hatten. Folgen Sie den Installationsanweisungen. Die Installation kann einige Zeit in Anspruch nehmen. Eventuell müssen Sie den Rechner neu starten.
