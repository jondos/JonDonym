Datenspuren im Internet verhindern: Mozilla SeaMonkey

Um ein Deanonymisieren zu vermeiden, während Sie mit SeaMonkey über JonDo surfen, sollten folgende Einstellungen vorgenommen werden. Öffnen Sie die Einstellungen, indem Sie im Menü den Punkt Bearbeiten -> Einstellungen wählen.
  • Cookies
    Klappen Sie den Punkt Datenschutz & Sicherheit in der linken Liste auf und wählen Sie Cookies. Wählen Sie Alle Cookies akzeptieren und Nur für aktuelle Session akzeptieren. Dadurch können Sie alle Seiten besuchen, die zwingend Cookies voraussetzen, können jedoch durch Cookies nicht wiedererkannt werden.

  • Java
    Wählen Sie den Punkt Erweitert in der Liste links. Haken Sie Java aktivieren ab.

  • JavaScript
    Wählen Sie den Punkt Skripte & Plugins, welcher ein Unterpunkt von Erweitert ist. Es wird empfohlen, JavaScript komplett zu deaktivieren. Wählen Sie dazu den Punkt Navigator ab.

Test: Unterziehen Sie diese Einstellungen einem Anonymitätstest.

Hinweis: Das sind lediglich grundlegende Einstellungen. Die Verwendung eines Filterproxys ermöglicht mehr Einstellungen zum Blockieren von Inhalten.
