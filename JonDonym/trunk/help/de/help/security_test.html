Test der Browsereinstellungen

Mithilfe verschiedener Testseiten können Sie erkennen, ob Ihr Browser richtig konfiguriert wurde. Empfehlenswert ist besonders der

JonDos-Anonymitätstest

Diese Seite gibt Ihnen einen schnellen Überblick über alle IP-, Header- und sonstigen Informationen, die Ihr Browser an Webseiten sendet. Alle Felder sollten grün gefärbt oder farblos sein. Orange ist normalerweise tolerierbar, Rot könnte verheerend für Ihre Anonymität sein.

Wenn Sie Rote Farben sehen, gehen Sie bitte die Anleitung zur Browservorbereitung nochmals gründlich durch. Überprüfen Sie vor allem die angezeigte IP-Adresse. Sofern diese Rot erscheint, bewegen Sie sich komplett ohne den Schutz durch JonDo oder Tor im Netz. Sorgen Sie außerdem dafür, dass Ihr Browser ActiveX, Java und Flash nur nach Ihrer jeweils einzelnen Bestätigung ausführen kann. Eine automatische Ausführung dieser Plugins würde der Webseite in den meisten Fällen Ihre IP-Adresse und wichtige Systemeinstellungen verraten und damit Ihre Anonymität gegenüber dieser Seite komplett oder zumindest weitestgehend aufheben.

Bitte beachten Sie, dass derzeit nur der JonDoFox-Browser alle im Test angegebenen Kriterien für sicheres und anonymes Surfen erfüllt. Bei anderen Browsern ist eine entsprechende Konfiguration, wenn überhaupt, nur für Experten möglich.

Weitere Testseiten:

  • Browserspy enthält eine große Liste mit Identifizierungsmerkmalen und ausgezeichneten Tests
  • Leader.ru gibt einen Überblick über verschiedene Identifizierungsmerkmale
  • Gurushaven.de sehr ausführliche deutsche Beschreibung von verschiedenen Tests
