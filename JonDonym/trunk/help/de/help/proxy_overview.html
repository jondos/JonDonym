Überblick über verschiedene Proxyarten

Die Wikipedia hat einen Artikel mit einer Auflistung von Proxyarten.
JonDo ist ein Anonymitätsproxy, der lokal auf dem Rechner installiert wird. JonDo selbst ist eine Art Zugangsprogamm zu den Mix-Kaskaden. Durch die Nutzung dieser Mix-Kaskaden wird die Zuordnung von Benutzer zu IP verschleiert, dadurch kann sich der Benutzer anonym im Netz bewegen.
Die Mix-Kaskaden sind nicht lokal auf dem Rechner installiert, sondern bestehen aus mehreren im Internet erreichbaren Servern. Sie ver- und entschlüsseln Nachrichten, mixen sie (daher der Name) und verändern das Aussehen der Nachrichten, damit sie aus Angreifersicht keinem Benutzer zugeordnet werden können.
Proxomitron ist ein Filterproxy, der ebenfalls lokal auf dem Rechner läuft. Durch den Einsatz von Filtern können z.B. Werbung aber auch schädliche JavaScripts, Java Applets, Cookies und HTTP Header gefiltert werden. Dazu schreibt Proxomitron die aufgerufene Webseiten in Echtzeit um, bevor sie an den Browser weitergeben werden.
