Bezahlung

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Bezahlung.
In diesem Dialog können Sie alle Einstellungen vornehmen, die Sie benötigen, um mit einer kostenpflichtigen Kaskade zu surfen.

Konten

In der Liste sehen Sie die Konten, die Sie bereits angelegt haben. Wenn Sie diesen Dialog zum ersten Mal aufrufen, wird die Liste leer sein.

Neben der Liste mit den Konten sehen sie das Erstellungsdatum, das Datum der letzten Kontostandsaktualisierung und das Gültigkeitsdatum des markierten Kontos. Nach Ablauf des Gültigkeitsdatums können Sie das Konto nicht mehr verwenden. Zum erneuten Einzahlen müssen Sie dann ein neues Konto anlegen.
Wenn neben der Liste der Hinweis "Keine Sicherheitskopie vorhanden" angezeigt ist, haben Sie von dem gewählten Konto noch keine Kopie erstellt. Ein Klick auf den Hiweis hat den gleichen Effekt wie ein Klick auf den Knopf "Exportieren".

Sie können folgende Aktionen durchführen:

  • Konto erzeugen
    Hiermit legen Sie ein neues Konto an. Der Kontoerstellungsprozess kann etwas Zeit in Anspruch nehmen, da hierfür ein aufwändiger kryptographischer Vorgang gestartet werden muss. Während des Erstellungsvorgangs werden Sie möglicherweise aufgefordert, eine Sicherheitsfrage zu beantworten. Bitte folgen Sie dazu den Anweisungen am Bildschirm. Wenn Ihr Konto erfolgreich erstellt wurde, erscheint es in der Liste der verfügbaren Konten.

  • Transaktionen
    Hier können Sie sich die letzten Aufladetransaktionen Ihrer Konto ansehen.

  • Passwort ändern
    Durch Klicken dieser Schaltfläche können Sie ein neues Passwort für die Verschlüsselung Ihrer Kontodaten angeben. Das Passwort gilt für alle Konten.
    Wenn Sie ihr Passwort löschen wollen, gehen Sie auf Passwort ändern und geben für das neue Passwort einfach nichts ein.

  • Importieren
    Wenn Sie diese Aktion wählen, können Sie ein zuvor exportiertes Konto wieder importieren.

Informationen zum Kontostand

Hier sehen Sie, mit wie vielen Bytes Sie Ihr Konto bereits insgesamt aufgeladen haben. Außerdem wird Ihnen der bisherige Gesamtverbrauch und der daraus resultierende aktuelle Kontostand angezeigt.

Sie können folgende Aktionen ausführen:

  • Aktivieren
    Mit diesem Knopf aktivieren Sie das gewählte Konto. Das Konto wird bei der Verbindung zu einer kostenpflichtigen Kaskade zum Bezahlen genutzt. Beim Wechsel des aktiven Kontos muss erst die Verbindung zur Kaskade getrennt werden. Das aktivierte Konto ist fett hervorgehoben.
    Sollten Sie ihre Konten passwortgeschützt haben und dieses Passwort beim Starten nicht eingegeben haben, so können sie ihre Konten durch diesen Knopf entsperren, sofern sie das richtige Passwort kennen.

  • Aktualisieren
    Wenn Sie diese Aktion wählen, wird der aktuelle Kontostand von der Bezahlinstanz abgefragt.

  • Löschen
    Hiermit löschen Sie Ihr Konto.
    ACHTUNG: Sollte sich auf Ihrem Konto noch Guthaben befinden, ist dieses unwiederbringlich verloren, wenn Sie keine Sicherheitskopie angelegt haben!

  • Exportieren
    Dieser Knopf speichert Ihre Kontodaten in einer externen Datei. Dies ist sinnvoll, um bei Datenverlust Ihrer Festplatte oder Neuinstallation von JonDo oder des Gesamtsystems wieder Zugriff auf Ihr Konto erhalten zu können. Sie sollten die Exportdatei also auf ein externes Medium (z.B. USB-Stick) speichern. Es wird empfohlen, ein neu angelegtes Konto zur Sicherung zu exportieren.
