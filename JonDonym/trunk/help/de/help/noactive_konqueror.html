Vermeiden von Datenspuren im Internet: Konqueror

Klicken Sie im Menü Einstellungen auf Konqueror einrichten. Folgendes Fenster zeigt sich:

Wählen Sie den Eintrag Java & JavaScript. Deaktivieren Sie Java global aktivieren. Wechseln Sie zum Tab JavaScript und deaktivieren Sie JavaScript global aktivieren.

Wählen Sie nun den Eintrag Cookies. Überprüfen Sie die Einstellungen in folgendem Fenster:


Folgende Punkte sollten aktiviert sein:
  • Cookies zulassen
  • Nur Cookies vom erstellenden Server annehmen
  • Sitzungs-Cookies automatisch annehmen
  • Standardregelung: Um Bestätigung bitten.

Darüberhinaus ist es empfehlenswert, unter dem Eintrag Plugins die Plugins zu deaktivieren.

Ergänzung: Für die genannten Einstellungen können Sie jeweils domainspezifische Ausnahmen einstellen. So können Sie Konqueror für bestimmte, vertrauenswürdige Seiten erlauben, JavaScript zu interpretieren oder Cookies anzunehmen.

Hinweis: Das sind lediglich grundlegende Einstellungen. Die Verwendung eines Filterproxys ermöglicht mehr Einstellungen zum Blockieren von Inhalten.

Test: Unterziehen Sie diese Einstellungen einem Anonymitätstest.
