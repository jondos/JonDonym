User Interface

To get to this panel, press button Settings in JonDo's main window, then choose the entry User Interface from the list to the left, then open the tab General.
In this dialog, you can make changes to the user interface appearance, language, and message displaying behavior of JonDo.

Look-and-Feel settings

  • Select Look-and-Feel
    JonDo supports various forms of graphic appearance for the program windows and user interface elements. Depending on the operating systems, several options are available.
    • Select language
      Here you can change the language in which the user interface elements appear.
      Here, too, a restart of JonDo is needed for the changes to take effect.

    • LookAndFeel
      Choose your preferred look and feel from the dropdown list. These changes require an instantaneous restart of JonDo to take effect.
      You may add new look and feel options by downloading some from the internet and importing them to JonDo. You need no packages created specifically for JonDo, instead you may use every look and feel package suitable for java.
      Such packages can be found at www.javootoo.com. After downloading a package, klick Import to add this look and feel.
      • Delete
        This will delete the selected look and feel. You can only delete those you added by yourself. Deleting the currently active look and feel is also not possible. The deleted look and feel will remain in memory until a restart of JonDo is performed.

      • Import
        A click will open a file selection window. Navigate to the location of the newly downloaded look and feel package and choose the ".jar" file.

    • Font size
      Here you can change the font size of the user interface elements.
      These changes have immediate effect.

View of JonDo

  • Extended View
    Extended View shows all options in the main window and in the configuration dialog.

  • Simplified View
    Simplified View will not show some advanced options. The functioning of JonDo is not changed by this, however.

  • Minimized view always on top
    Sets the JonDo mini window always on top of all other windows of the operating system. If unchecked, the mini window will hide behind other windows.

Program Start

  • After start of JonDo...
    If you activate this option, there are two further possible options:

    • ...show the minimal view
      This option minimizes the main JonDo window immediately after starting.

    • ...move JonDo into the Systray
      If this option is chosen, the main JonDo window moves to the Windows systray immediately after starting.

Remember location

This option is only available in Extended View.

If you put a check in one of the following boxes, the corresponding JonDo window will appear at the previous position when you close and restart JonDo.

  • Main window
  • Configuration window
  • Iconified window
  • Help window

Remember size

This option is only available in Extended View.

  • Help window
    Checking this box will make JonDo remember the size of the help window.

Program shutdown

This option is only available in Extended View.

  • Show warning when closing JonDo
    When checked, JonDo displays a warning which says that your browser's proxy settings need to be reconfigured in order to continue surfing the web.
