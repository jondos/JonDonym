Supported programs

JonDo currenty only supports the following internet protocols:

  • HTTP
  • HTTPS
  • FTP (download only)
  • Gopher

These are typical programs supported by JonDo:

  • Webbrowsers
  • Download-Managers (insert JonDo as HTTP proxy like in web browsers)

JonDo is mainly designed for anonymous web browsing. Therefore, the following applications are not supported by JonDo:

  • Filesharing, Peer-To-Peer (Emule, BitTorrent, BitComet,...)
  • Chat, Instant Messaging (ICQ, AIM, Yahoo Messenger,...)
  • E-Mail
  • Internet telephony (VoIP, Skype,...)
  • FTP, Telnet, SSH,...

Nevertheless, you find hints on the net that explain how to anonymise the mentioned applications with JonDo. They are kind of alchemy and do not work really. Some of these connections, like Skype, may be run over JonDo theoretically, but this does not anonymise them, only the traffic is forwarded. The Skype operator himself can deanonymise each of the communication connections with ease, despite the Mixes in the middle.

Moreover, script based (JavaScript/Java/Flash) web applications cannot be anonymized, like Chatrooms, interactive maps and online games. Scripts can circumvent JonDo's security mechanisms more or less easy. Whoever wants to be anonymous should not use web applications of this kind.
