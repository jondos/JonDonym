Avoid data trails on the internet: Mozilla Firefox

You may do settings for firefox manually or use a plugin.

Manually

Open Options in Menu Tools.
  • Cookies
    Choose Tab Privacy and activate Allow sites to set Cookies. At Keep Cookies:, choose the option until I close Firefox from the drop-down menu. This way you won't have trouble with any sites since you accept them all, but you cannot be tracked.

  • Java & JavaScript
    Choose Tab Content. Unselect Enable Java and Enable JavaScript.

Plugin: Using PrefBar

If you want to use PrefBar to manage these settings comfortably, please read this this guide.

Note: These are basic settings. Using a filter proxy will offer more options for blocking content.
Cookie Culler is a plugin which allows deletion of unwanted cookies while keeping wanted cookies.
