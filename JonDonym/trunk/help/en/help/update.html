JonDo Update

Note: This panel is only available in Extended View.
To get to this panel, press button Settings in JonDo's main window, then choose the entry JonDo Update from the list to the left.

Installed Version

  • Version
    Here you see the version number of the JonDo program that is installed on your system.

  • Date
    This field shows the timestamp of the currently installed version.

  • Type
    The type of the currently installed version is displayed here. There are two possible types: The "Release Version" has been thoroughly tested and includes only the functions that should have no bugs or conflicts. The "Development Version" includes all functions that are available, even if they are only partially implemented or contain bugs or mistakes.
    For most users, we recommend the "Release Version".

Latest Version

This area contains the same fields as the "Installed Version" area. The information in these fields also corresponds to the fields in the "Installed Version" area. However, the information does not refer to the software currently installed on your system but rather refers to the JonDo version that can currently be downloaded. Using the drop-down menu "Type", you can choose whether the information for the "Release Version" or "Development Version" is displayed.

Options

  • Allow non-anonymous update
    It is recommended to activate this option. Thereby JonDo can perform updates even if you have no anonymized connection currently established.
    If this option is deactivated, updates may be done via anonymized connections only.

  • Show information window if new JonDo version is available
    If this option is active, an information window will be shown whenever a new version of JonDo is available.
    Even if this option is not active, there will appear a small hint on the main window anyway. JonDo checks for new versions on startup and then every 12 hours.

  • Show information window if new version of Java (JRE)is available
    If this option is active, an information window will be shown whenever a new version of Java is available. In any case, a short reminder is shown in he main window.
    JonDo checks for new versions on startup and every 12 hours. Upgrading JRE has to be done manually. More...

Info

This field displays information about the update's installation progress as the update is being installed.
Furthermore, it shows the result of checking for a new version.

Other Elements

  • Check for new updates
    Click this button to check if a newer version than the currently installed version is available for download.

  • Upgrade...
    With this button, you can download the latest version of JonDo from the internet and install it on your system.
