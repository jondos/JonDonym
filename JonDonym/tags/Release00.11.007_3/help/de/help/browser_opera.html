Proxy Einstellungen: Opera

JonDos empfiehlt den bereits fertig konfigurierten JonDoFox zum sicheren, anonymen Websurfen. Er ist als vollständiger Browser oder als Firefox-Profil für alle Betriebssysteme verfügbar.

Um JonDo stattdessen in Opera einzurichten gehen Sie folgendermaßen vor:

  1. Wählen Sie im Menü Extras das Untermenü Einstellungen. Danach öffnet sich das Einstellungsfenster. Wählen sie in diesem den Punkt Netzwerk/Verbindung... :

  2. Nun wird der Proxyserver eingestellt. Dazu klicken Sie auf die Schaltfläche Proxyserver. Jetzt öffnet sich ein Fenster, in dem Sie folgende Einstellungen vornehmen müssen: Setzen Sie das Häkchen vor HTTP, HTTPS und FTP und geben Sie jeweils in das leere Feld dahinter die Serveradresse 127.0.0.1 ein. Als Port wird 4001 eingetragen.

    Wollen Sie Adressen festlegen, für die JonDo nicht verwendet werden soll, so können Sie im Feld Proxyserver für folgende Adressen nicht benutzen: Ausnahmen festlegen. Aktivieren Sie die Ausnahmeregeln, indem Sie in dieses Feld den Haken dazu setzen.

    Hinweis:
    Falls Sie im JonDo einen anderen Listenerport eingestellt haben, müssen Sie diesen anstatt 4001 eintragen.

Geschwindigkeitsoptimierung

Folgende Konfigurationsänderungen werden zusätzlich empfohlen, damit Opera ohne unnötige Verzögerungen beim Laden von Webseiten mit JonDo zusammenarbeitet:

  • In der Adresszeile eingeben: opera:config + Enter (bei eingeschaltetem Javascript).
  • Dann scrollt man runter zu *Performance*.
  • Die *Network Buffer Size* auf 256 stellen, Haken raus bei *Reduce Max Persistent HTTP Connections*. Speichern. Dann noch zu *Proxy* scrollen und einen Haken setzen bei *Enable start with HTTP 1.1 for proxy*. Speichern. Opera beenden und wieder neustarten.
  • Dann noch in Opera oben bei Extras, Einstellungen, Erweitert, bei *Netzwerk* die *Max. Verbindungen zum Server* auf 16 stellen und die *Max. Verbindungen gesamt* auf 20. Etwas darüber noch auf "Proxyserver..." klicken und HTTP 1.1 für Proxy-Verbindungen einschalten.

Diese Optimierung wurde getestet mit Opera 9.63.10476.
