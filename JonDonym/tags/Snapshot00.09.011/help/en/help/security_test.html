Checking settings of browser / filter

We will use a test site which lets us know whether your browser (and filter proxy, if any) is correctly configured. JonDos suggests the preconfigured, portable JonDoFox for secure anonymous web browsing. It is availabe as complete browser (Windows) or as Firefox profile for all operating systems and passes all tests.

JonDos anonymity test

The site gives a quick overview of all IP, header and other information that your browser sends to websites. All fields should be colored green or be without color, orange would be tolerable, red may be a killer for your anonymity. If you see orange or red colors, move your mouse pointer above the respective fields, and you will see hints and an explanation as popups.

In case the results of the test do not meet requirements this might be due to the following reasons:

  • JonDo has not been installed correctly
  • Neither JonDoFox nor filter proxy has been used, or you did not set the data protection configuration for your browser correctly .
  • Filter proxy is not using correct filter settings.
  • Filter proxy is not using JonDo as external proxy.
  • Your browser is not using the filter proxy as proxy.
