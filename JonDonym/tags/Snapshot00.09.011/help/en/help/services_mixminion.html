Services: Mixminion

This panel is only available in Extended View. (Visible in the developer version of JonDo only)

To get to this panel, press button Settings in JonDo's main window, then choose the entry Services from the list to the left, then open the tab Mixminion. In this tab, you can find information and settings for the Mixminion anonymization system.
If you would like to use JonDo to access the Mixminion system, you will need to enter JonDo as a proxy on port 25 in your email client.

  • Activate
    Using Mixminion with JonDo is still experimental. Deactivating this checkbox will result in prevention of accidental usage of Mixminion as well as less memory usage by JonDo.

  • Available Mixminion Nodes:
    This tab shows other computers that are a part of the Mixminion network and offer a connection to the Mixminion network.

  • Fetch list:
    This button retrieves an updated list of currently accessible Mixminion nodes.

  • Preferences for anonymous routing:

    • Nodes per connection:
      Here you can set the number of Mixminion nodes through which a connection should be routed before being connected to the destination computer. The more nodes you use, the more effective the anonymization of the connection.

  • Email settings for answer mails:

    • My email address:
      You may enter your email address which will be used for answer mails. If you do not specify an address, JonDo will use the address from your email client.
