Services:General

This panel is only available in Extended View.

To get to this panel, press button Settings in JonDo's main window, then choose the entry Services from the list to the left, then open the tab General
.

  • Assignment of internet services to anonymizer services
    This overview shows which internet service is assigned to which anonymizer service. Future releases of JonDo will allow you to set your own assignments. Meanwhile, this is merely for your information.

  • Deny all data traffic without call back if anonymous mode is switched off
    If this box is checked, JonDo will not transfer non-anonymous traffic. Your browser will show a hint which says that browsing the web is currently not possible.
    If you do NOT check this box, JonDo will show a warning window which has the option to allow non-anonymous browsing for a short amount of time.
    More information

  • Connect to explicitly unlocked cascades only (Whitelist)
    If activated, JonDo will only allow connections to cascades that you have explicitly checked and unlocked before. This optimally protects you from connecting to untrusted JonDonym services. On the other hand, you will not automatically benefit from new available services, but will have to unlock them one by one in the JonDonym service view.

  • Auto connect after program start
    If this box is checked, JonDo automatically connects to the anonymization service when JonDo is started.

  • Auto re-connect after connection to mix was lost
    Check this box if you want JonDo to automatically reconnect to the anonymization service if the connection is lost.

  • Automatically change Mix cascades
    With this option activated, JonDo will choose a cascade by chance. During anonymous browsing, however, cascades will only be switched if the active cascade is offline.

  • Send dummy packet
    Here you can decide if fake packets without any content should be sent and if so, how often. This simulates continuous communication for any potential observers.

  • Login timeout
    If your internet connection is too slow, a too short timeout for the login to services may lead prevent JonDo from establishing an anonymous connection. A too high timeout may cause long waiting times when connecting if the currently chosen service is slow or not available. Therefore set the timeout lower or higher as you need.
