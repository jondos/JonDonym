Debugging

This panel is only available in Extended View.

To get to this panel, press button Settings in JonDo's main window, then choose the entry Debugging from the list to the left.
Here you can configure how many status and error messages appear at the console while running JonDo (for example a DOS window console). If you're having problems with JonDo, you might be able to localize them using these error messages. If you're unable to find the solution and request help from us, you should definitely attach the appropriate error messages to your e-mail.
In that case, please attach your JonDo configuration file also.

Show messages belonging to the following subsystems
By setting the checkboxes for the appropriate topics, you can choose whether you want to receive messages concerning that topic.

Show messages of the following level
With the sliding bar, you can choose the level of importance a message must have in order to be shown on the console. If it's at the very bottom, only messages that concern severe problems that prevent correct functioning of the service will be shown.

Detail level of log messages
The higher the level, the more detailed log messages will become.

Please read the troubleshooting page as well.

Show messages in a separate window
If this box is checked, a separate window will appear for debug messages. Within this window, you can copy or save the messages and send them to us by e-mail.

Write messages to the following file
If this box is checked, you can indicate a file in which the debug messages are to be saved.
