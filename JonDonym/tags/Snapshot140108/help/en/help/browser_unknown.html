Proxy settings for browsers: Other Browsers

JonDos suggests the preconfigured, portable JonDoFox for secure anonymous web browsing. It is availabe as complete browser (Windows) or as Firefox profile for all operating systems.

However, it's theoretically possible to set up JonDo for any web browser that supports surfing over an HTTP proxy. You simply have to look in the settings menu of your browser for the possibility of configuring a proxy server and enter the following:

  • HTTP proxy address: 127.0.0.1
  • HTTP proxy port: 4001
You should be able to specify addresses where JonDo shall not be used somewhere around these settings.
Please note that such exceptions can be managed in a more flexible way using Proxomitron.
Note: If you set a different listener port in JonDo, you'll have to enter this port here too, rather than 4001.
