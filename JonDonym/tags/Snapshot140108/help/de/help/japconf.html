JonDo Konfigurationsdatei

JonDo erstellt eine Konfigurationsdatei, welche die gesamten Einstellungen beinhaltet.
Es kann nützlich sein, den Pfad zu der Konfigurationsdatei zu kennen, für den Fall dass Sie ...
  • die Datei einschicken wollen für Debugging-Zwecke
  • die Datei löschen möchten, wenn JonDo fehlerhaft konfiguriert wurde
  • die Datei auf einen anderen Rechner kopieren wollen
Die Datei liegt unter verschiedenen Betriebssystemen an verschiedenen Orten:
  • Windows
    C:\Dokumente und Einstellungen\<username>\jap.conf
  • Linux
    <userhome>/.jap.conf
  • Mac
    <userhome>/Library/Preferences/jap.conf
