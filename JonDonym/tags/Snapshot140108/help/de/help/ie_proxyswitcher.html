Proxy Einstellungen für den Internet Explorer komfortabel ändern

Proxy Switcher Lite

Dieses Zusatzprogramm (nur für Windows) kann verschiedene Proxy Einstellungen verwalten und zwischen ihnen umschalten. Somit bleibt einem das Umstellen im Internet Explorer erspart.

  • Installation

    Laden Sie Proxy Switcher von dieser Seite herunter. Es ist nicht notwendig, die Standard Version zu kaufen, die kostenlose Version reicht aus.
    Doppelklicken Sie auf die heruntergeladene Datei, um die Installation zu starten. Folgen Sie den Installationsanweisungen.
    Nach der Installation ist Switch Proxy in der System Tray mit dem Icon zu finden.

  • Einstellungen

    Klicken Sie auf das Icon. Ein kleines Fenster öffnet sich, klicken sie dort auf das Symbol .
    Nun öffnet sich folgendes Fenster:


    Tragen Sie die Adresse "127.0.0.1" und den Port "4001" ein. Klicken Sie auf OK. Damit erscheint ein neuer Eintrag in der Liste. Wenn Sie diesen Eintrag aktivieren wollen (also über JonDo surfen wollen), dann klicken Sie auf . Zum Deaktivieren (Surfen ohne JonDo) klicken Sie auf
    Hinweis: Falls Sie im JonDo einen anderen Listenerport eingestellt haben müssen Sie diesen anstatt 4001 eintragen.
