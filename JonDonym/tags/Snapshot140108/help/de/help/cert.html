Zertifikatsstellen

Dieses Panel steht nur in der erweiterten Ansicht zur Verfügung.

Um zu diesem Einstellungsfenster zu kommen, klicken Sie im Hauptfenster von JonDo auf Einstellungen und wählen anschließend aus der linken Liste den Eintrag Zertifikatsstellen.

Ein Zertifikat ist ein von einer Zertifizierungsinstanz (z.B. dem JonDo-Team) digital unterschriebener (digitaler) Behälter für einen öffentlichen Verschlüsselungsschlüssel, der gegenüber Dritten (z.B. den Nutzern) durch die Signatur bestätigt, dass dieser Schlüssel einer ganz bestimmten Person oder Organisation gehört. Gäbe es diese Bestätigung für die Zertifikate nicht, so könnte sich z.B. jeder als JonDo-Team oder als vertrauenswürdiger Mixbetreiber ausgeben und etwa Honeypot-Mixe betreiben, oder sich ganz einfach in die Kommunikation zwischen Nutzern und tatsächlich vertrauenswürdigen Mixen/Betreibern einschalten und sämtliche Daten mitlesen. Auch das Einspielen falscher Update-Nachrichten wäre leicht möglich. (Weitere Informationen zu Zertifikaten...)

  • Ich will nur von folgenden Stellen zertifizierte Server verwenden:
    Diese Liste enthält alle Zertifizierungsstellen, denen Sie vertrauen. Wird ein Server nicht von einer Stelle, die in dieser Liste enthalten ist, zertifiziert, so wird JonDo keine Verbindung mit diesem Server aufbauen.

  • Importieren:
    Mit dieser Schaltfläche fügen Sie Zertifizierungsstellen hinzu. Im erscheinenden Dateidialog muss das entsprechende Zertifikat ausgewählt werden.

  • Entfernen:
    Hiermit löschen Sie den markierten Eintrag aus der Liste.

  • Deaktivieren:
    Durch Klicken dieses Knopfes schalten Sie den markierten Eintrag vorübergehend in den Status "nicht vertrauenswürdig".

  • Details...
    Durch Klicken dieses Knopfes können Sie das Zertifikat des gewählten Eintrages ansehen.
